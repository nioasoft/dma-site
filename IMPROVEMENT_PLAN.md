## סקירה כללית

### מה האתר הזה
DMA הוא אתר של חברה המתמחה בתשתיות מתח נמוך, תקשורת ואבטחה לבתי יוקרה ועסקים. קהל היעד הוא בעלי בתים אמידים, אדריכלים/מעצבים, ועסקים.

### מה עובד טוב
- עיצוב יוקרתי ומקצועי
- 12 מאמרי בלוג איכותיים עם תוכן חינוכי מעולה
- מיצוב ברור (יוקרה, דיסקרטיות, איכות)
- עדויות אמינות עם פרטים ספציפיים
- שפה משכנעת ומקצועית

### מה דורש שיפור
- טופס יצירת קשר לא עובד
- חסרים עמודי שירותים מפורטים
- חסר מידע על תמחור/תהליך עבודה
- SEO לא מותאם
- חסרים עמודי נחיתה ממוקדים

### 4. חסר sitemap.xml
**ליצור קובץ:** `/src/app/sitemap.ts`
```typescript
import { blogPosts } from '@/data/blogPosts';
import { MetadataRoute } from 'next';

export default function sitemap(): MetadataRoute.Sitemap {
    const baseUrl = 'https://dma.co.il';

    const blogUrls = blogPosts.map((post) => ({
        url: `${baseUrl}/blog/${post.slug}`,
        lastModified: new Date(post.date),
        changeFrequency: 'monthly' as const,
        priority: 0.7,
    }));

    return [
        {
            url: baseUrl,
            lastModified: new Date(),
            changeFrequency: 'weekly',
            priority: 1,
        },
        {
            url: `${baseUrl}/about`,
            lastModified: new Date(),
            changeFrequency: 'monthly',
            priority: 0.8,
        },
        {
            url: `${baseUrl}/services`,
            lastModified: new Date(),
            changeFrequency: 'monthly',
            priority: 0.9,
        },
        {
            url: `${baseUrl}/blog`,
            lastModified: new Date(),
            changeFrequency: 'weekly',
            priority: 0.8,
        },
        {
            url: `${baseUrl}/testimonials`,
            lastModified: new Date(),
            changeFrequency: 'monthly',
            priority: 0.7,
        },
        {
            url: `${baseUrl}/contact`,
            lastModified: new Date(),
            changeFrequency: 'yearly',
            priority: 0.6,
        },
        ...blogUrls,
    ];
}
```

## עמודים חסרים - להוספה

### 1. עמוד FAQ (שאלות נפוצות)

**ליצור קובץ:** `/src/app/faq/page.tsx`

**תוכן מוצע:**

```markdown
# שאלות נפוצות

## כללי

### מה ההבדל בין DMA לחברות התקנה אחרות?
אנחנו לא רק "מתקינים" - אנחנו מתכננים. כל פרויקט מתחיל בניתוח הנדסי מעמיק
של תוכניות האדריכלות, הבנת הצרכים הנוכחיים והעתידיים, ותכנון תשתית שתחזיק
לעשורים. אנחנו עובדים עם ציוד ברמה צבאית/תעשייתית בלבד, ומחויבים לרמת גימור
שלא רואים בארץ.

### האם אתם עובדים רק עם בתי יוקרה?
לא. אנחנו עובדים עם מגוון רחב של לקוחות - מדירות קטנות שרוצות רשת WiFi
יציבה, דרך בתים פרטיים שמחפשים בית חכם מלא, ועד מפעלים ורשתות עסקיות
בפריסה ארצית. העיקרון שלנו זהה בכל הפרויקטים: עבודה מקצועית ללא פשרות.

### מהו טווח המחירים שלכם?
התמחור שלנו מותאם לפרויקט. פגישת ייעוץ ראשונית היא ללא עלות, ובעקבותיה
נספק הצעת מחיר מפורטת. להתרשמות כללית:
- סקר תשתיות לבית קיים: החל מ-1,500 ש"ח
- מערכת WiFi מקצועית (בית 200 מ"ר): 8,000-15,000 ש"ח
- מערכת אבטחה בסיסית (4-8 מצלמות): 15,000-35,000 ש"ח
- בית חכם מלא: החל מ-80,000 ש"ח
*המחירים להמחשה בלבד - כל פרויקט מתומחר באופן אישי.

### כמה זמן לוקח פרויקט טיפוסי?
- בית חדש בבנייה: אנחנו נכנסים בשלב השלד ומלווים עד לאכלוס (3-12 חודשים)
- שיפוץ/התקנה בבית קיים: 2-8 שבועות בהתאם להיקף
- מערכת אבטחה בסיסית: 1-3 ימים
- רשת WiFi מקצועית: 1-2 ימים

## תשתיות ותקשורת

### למה אני צריך סיבים אופטיים בבית?
סיבים אופטיים הם ה"כביש המהיר" של הנתונים. הם מאפשרים מהירויות של עד
10Gbps (פי 100 מחיבור ביתי סטנדרטי), ללא הפרעות חשמליות, ועמידים
לשנים רבות. בבית גדול (מעל 200 מ"ר) או בית עם קירות בטון עבים, סיבים
אופטיים הם ההבדל בין רשת שעובדת לרשת שמתקתקת.

### מה ההבדל בין WiFi ביתי לרשת מנוהלת?
ראוטר ביתי רגיל מיועד לדירה קטנה. ברגע שיש קירות בטון, מרחקים גדולים,
או הרבה מכשירים מחוברים - הוא קורס. רשת מנוהלת (Enterprise-grade)
כוללת מספר נקודות גישה מפוזרות, בקר מרכזי שמנהל את התעבורה, ו"נדידה"
חכמה שמעבירה אתכם מאנטנה לאנטנה בלי שתרגישו.

### האם WiFi-7 כבר רלוונטי?
כן ולא. אם אתם בונים או משפצים עכשיו, כדאי להתקין ציוד WiFi-7 כי הוא
יהיה הסטנדרט לשנים הקרובות. אבל אם יש לכם רשת עובדת - אין צורך לרוץ
לשדרג. אנחנו תמיד בוחרים את הציוד המתאים לצרכים האמיתיים שלכם.

## אבטחה

### האם מצלמות אבטחה באמת מרתיעות פורצים?
מצלמות רגילות - פחות. פורצים מקצועיים יודעים שרוב המצלמות רק מקליטות,
ושעד שמישהו יראה את הסרטון הם כבר יהיו רחוק. לעומת זאת, מערכת
אנליטיקה אקטיבית כמו שאנחנו מתקינים - שמפעילה תאורה, כריזה והתראות
בזמן אמת - מרתיעה מאוד. ב-99% מהמקרים, הפורץ בורח עוד לפני שנגע בדלת.

### מה זה אנליטיקה מבוססת AI?
זו תוכנה חכמה שמנתחת את הווידאו בזמן אמת ויודעת להבדיל בין אדם, רכב,
ובעל חיים. במקום לקבל 50 התראות שווא ביום על חתולים ושקיות ניילון,
אתם מקבלים התראה אחת כשבאמת יש אדם זר בחצר - ובזמן אמת, לא למחרת בבוקר.

### האם זה חוקי להתקין מצלמות בבית?
כן, אבל יש כללים. מותר לצלם את השטח הפרטי שלכם (חצר, כניסה, פנים הבית).
אסור לצלם רשות הרבים או בתים של שכנים. אנחנו מייעצים ללקוחות איפה
למקם מצלמות כדי לקבל כיסוי מקסימלי בצורה חוקית.

### מה קורה אם נפרץ לי לרשת ויראו את המצלמות שלי?
זה בדיוק למה אנחנו בונים רשתות מאובטחות. המצלמות שלנו יושבות על רשת
נפרדת (VLAN) שמבודדת מהאינטרנט הרגיל. הגישה מרחוק מתבצעת דרך VPN
מוצפן, וכל הציוד מוגדר עם סיסמאות חזקות. אנחנו לא משתמשים בסיסמאות
ברירת מחדל של היצרן - זו הסיבה העיקרית לפריצות.

## בית חכם

### האם בית חכם זה רק גימיק?
תלוי מי התקין אותו. בית חכם שהותקן נכון מפשט את החיים: לחיצה אחת
מכינה את הבית לשינה (כיבוי אורות, נעילת דלתות, הפעלת אזעקה), הווילונות
יורדים אוטומטית כשהשמש חזקה, והמזגן מתחיל לקרר לפני שאתם מגיעים הביתה.
בית חכם גרוע - זה כאב ראש אינסופי של אפליקציות שלא עובדות.

### איזה מותג בית חכם הכי טוב?
אין תשובה אחת. זה תלוי בתקציב, בגודל הבית, ובמה שחשוב לכם. אנחנו
עובדים עם מגוון מערכות - מ-KNX הגרמנית (הכי יציבה ויקרה) דרך Control4
(איזון טוב בין מחיר ליכולות) ועד Loxone (מצוין לפרויקטים בינוניים).
בפגישת הייעוץ נמליץ על הפתרון המתאים לכם.

### האם אני יכול לשלוט בבית מהטלפון כשאני בחו"ל?
בהחלט. כל המערכות שאנחנו מתקינים מאפשרות שליטה מלאה מכל מקום בעולם -
לראות מצלמות, לשלוט בתאורה ומזגנים, לפתוח שער לאורחים, ועוד. הגישה
מאובטחת ומוצפנת, כמובן.

## שירות ותמיכה

### מה קורה אם יש תקלה אחרי ההתקנה?
יש לנו מוקד שירות פעיל (*5555). ללקוחות עם הסכם SLA, אנחנו מתחייבים
לזמני תגובה מוגדרים (בין שעתיים ל-24 שעות בהתאם לחבילה). בנוסף, אנחנו
מנטרים את המערכות שלכם מרחוק 24/7 ולעתים קרובות מטפלים בבעיות עוד
לפני שאתם מרגישים בהן.

### האם אתם נותנים אחריות?
כן. אחריות על עבודה - שנה. אחריות על ציוד - בהתאם ליצרן (בדרך כלל
2-5 שנים). אנחנו גם מציעים הסכמי שירות שמרחיבים את האחריות וכוללים
ביקורות תקופתיות, עדכוני תוכנה, וטיפול עדיפותי בתקלות.

### האם אתם עובדים בכל הארץ?
כן. המשרד שלנו במרכז, אבל אנחנו נותנים שירות בפריסה ארצית - מקריית
שמונה ועד אילת. לפרויקטים גדולים בפריפריה, אנחנו מתאמים ימי עבודה
רצופים כדי למזער את ההגעות.
```

## שלב 2: סקר טכני ותכנון

### מה קורה?
אנחנו מבצעים סקר מעמיק של המבנה - מדידות, בדיקת תשתיות קיימות, מיפוי

יהיה יקר מאוד לתקן אחר כך.

### תיאום:
עובדים בצמוד עם הקבלן, החשמלאי והאדריכל.

## שלב 4: התקנת ציוד (שלב הגימור)

### מה קורה?
מתקינים את ציוד הקצה: מצלמות, נקודות גישה, בקרים, רמקולים, מתגים
