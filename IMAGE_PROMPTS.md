## עדכון הקוד אחרי הוספת התמונות

כשהתמונות יהיו מוכנות, צריך לעדכן את `page.tsx` להשתמש בהן במקום ה-placeholders:

```tsx
// בקובץ page.tsx, החלף את cardImage ב:
<div className={styles.cardImage}>
    <Image
        src={caseStudy.image}
        alt={caseStudy.title}
        fill
        style={{ objectFit: 'cover' }}
    />
</div>
```

וצריך להוסיף `import Image from 'next/image'` בראש הקובץ.
